Courses

The following courses are available:

  • Introduction to Perl
    One-day (lecture style) or two-day (with practical sessions) course assuming no previous knowledge of Perl
  • Intermediate Perl
    One-day (lecture style) or two-day (with practical sessions) course for people who have done some Perl but who want to really know the language
  • Advanced Perl Techniques
    One-day (lecture style) or two-day (with practical sessions) course bringing you up to date with the advanced techniques used by the world’s foremost Perl programmers
  • Web Programming with Perl
    One-day (lecture style) or two-day (with practical sessions) course on the latest techniques for building dynamic web applications using Perl
  • Introduction to Modern Perl
    One day overview of the latest Perl tools
  • Modern Core Perl
    Half-day overview of the most recent additions to the core Perl language
